## 📚 PART 1 — OPENCLAW MASTERY

### Official Docs (local on your machine)
All docs are installed at:
```
~/.nvm/versions/node/v24.14.1/lib/node_modules/openclaw/docs/
```

Open them anytime with:
```bash
openclaw dashboard
```
Then click the docs link in the UI.

🔗 **https://docs.openclaw.ai**

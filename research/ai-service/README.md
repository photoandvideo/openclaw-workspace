# Nexcom AI — nexcomai.ai

AI Assistant Service for local businesses in Clearwater, FL & beyond.

## Files

- `index.html` — Main landing page
- `demo.html` — Plumbing chatbot demo (static, no server)
- `pitch.html` — Printable pitch one-pager
- `realestate/` — Real estate chatbot + backend (Node.js)

## Deploy to Netlify (landing page)

1. Go to netlify.com → "Add new site" → "Deploy manually"
2. Drag and drop this folder (ai-service/)
3. Set custom domain: nexcomai.ai
4. Done!

## Run real estate demo locally

```bash
cd realestate
npm install
node server.js
```

Then open http://localhost:3001
